# OrdersGo

An order system which built in Node.js and Vue.

## Built With

* Backend - [Node.js](https://nodejs.org/)
* Frontend - [Vue](https://vuejs.org/)

### Port

Backend
* 3000 - Orders App
* 3030 - Payments App

Frontend
* 8080 - Orders Portal

RxJs is applied on frontend to keep track on database data.
